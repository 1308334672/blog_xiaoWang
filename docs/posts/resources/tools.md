# 🛠️ 常用开发工具

> 好的工具能让开发效率翻倍，整理了一些日常使用的神器。

## IDE & 编辑器

| 工具 | 说明 | 官网 |
|------|------|------|
| VS Code | 前端开发首选，插件生态丰富 | [code.visualstudio.com](https://code.visualstudio.com/) |
| IntelliJ IDEA | Java 开发神器 | [jetbrains.com/idea](https://www.jetbrains.com/idea/) |

## VS Code 推荐插件

```
Essential 插件：
├── ESLint           - JS 代码检查
├── Prettier         - 代码格式化
├── Volar            - Vue3 支持
├── GitLens          - Git 增强
├── Auto Rename Tag  - 自动同步标签名
└── Chinese (Simplified) - 中文界面
```

## 命令行工具

```bash
# nvm - Node.js 版本管理
nvm install 20
nvm use 20

# git 常用命令
git status
git add .
git commit -m "feat: 新功能描述"
git push origin main
```
